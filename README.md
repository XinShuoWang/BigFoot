# BigFoot
## 优点：
1. 跨平台：使用Java开发，原生跨平台。
2. 无依赖，可执行JAR包体积较小。
3. 支持自定义协议，用户可以根据封锁情况自定义协议进行加密，默认使用SSL加密。
4. 不易被检测：使用SSL加密，与合法流量融合。
5. 使用OSS存储PAC文件，云端更新，不用担心无法更新PAC文件。

### Windows
1. 设置PAC文件（PAC文件设置默认在1080监听本地端口）
2. 下载客户端
3. 配置JRE（Java Runtime Environment）
4. 执行程序
